Website ini dibuat spesial untuk Valentine dengan stack:
- React (Vite)
- TypeScript
- Tailwind CSS
- Framer Motion
- Canvas Confetti

## Cara Menjalankan

1. Install dependencies:
   ```bash
   npm install
   ```

2. Jalankan server development:
   ```bash
   npm run dev
   ```

3. Buka browser di `http://localhost:5173`

## Struktur Project
- `src/components/`: Komponen UI (Hero, LoveMessage, Gallery, dll)
- `src/App.tsx`: Main logic dan layout
- `src/index.css`: Tailwind directives dan custom styles

## Customisasi
- Ganti foto di `src/components/Gallery.tsx`
- Ubah pesan di `src/components/LoveMessage.tsx`
- Ubah alasan di `src/components/Reasons.tsx`
